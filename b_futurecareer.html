Web Developer

Ever since this summer, I have wanted to become a web developer. This interest sparked me over the summer when I was taking a coding class. The class assigned us to make a simple website. Since then, I have been making additions to this website. I love how I can personalize it to my liking.

Description

A web developer creates and builds websites. Web developers know various code language such as Java, Python, CSS, and so on. They make websites we use everyday to work. From buttons to texts to links to search bars, all are made by a developer. Without web developers, there wouldn't be any site to view! If a client needs a site to be made, they will contact a developer to create one for them. Some technical skills that are related to web developing would be: knowing HTML/ Java, write website applications, have experience in various programs such as Github and Photoshop, be detail-oriented, strong problem solver, and work well under pressure.

Education

Web developers should have an interest in computers and computer programming. They should also have good accuracy and attention to detail. To become a web developer, there are a few ways you can go. Many developers have a bachelor's degree in Programming, others major in Computer Science or Web Designing. Currently, I am taking Advanced Placement Computer Science Java at my school, and I enjoy that class. My plan for college is to attend Georgia Institute of Technology and major in Computer Science.

Job Outlook

According to the Bureau of Labor Statistics, the employment of web developers is to increase 15% from 2016 to 2026. As websites are accessed through almost all electronics (mobile phones, laptops, smart TVs, etc) the demand for new websites will be high.

Salary

The income of a web developer varies depending on how experienced one is. In Illinois, the annual income of an entry level employee would be around $50,000. However, an experienced employee earns around $90,000. You can either work as a full-time or part-time on a contract.
